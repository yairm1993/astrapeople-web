# AstraPeople - Web corporativa

Este repositorio contiene el sitio web corporativo de **AstraPeople**.

## Tecnologías

- HTML
- CSS
- JavaScript

No incluye PHP ni configuraciones de servidor. Es un sitio 100% estático, listo para deploy en cualquier hosting de estáticos (por ejemplo GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.).

## Estructura

```
.
├── index.html
├── css/
├── js/
├── images/
└── robots.txt
```

## Deploy

- **GitHub Pages**: configura la rama y carpeta raíz como fuente de publicación.
- **Hosting estático**: sube el contenido tal cual manteniendo la estructura de carpetas.
